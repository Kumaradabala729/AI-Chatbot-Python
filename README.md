# AI Chatbot using Python

This project is a web-based AI chatbot built using Python and Flask.

Features:
- Interactive chat interface
- AI-generated responses
- Web-based chatbot
- Built using Python, Flask, HTML, CSS, JavaScript

Technologies Used:
- Python
- Flask
- OpenAI API
- HTML
- CSS
- JavaScript

How to Run:

pip install -r requirements.txt
python app.py
